# Project

## First, set up the project env

```bash
git clone https://github.com/Vmiu/COMP4651-FP.git

cd COMP4651-FP

npm install
```

## Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
